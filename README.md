# Food Ordering App (MERN)

A full-stack food ordering web application inspired by modern food delivery platforms.  
Users can browse restaurants, explore menus, add items to a cart, place orders, and view their order history.
This project demonstrates a complete **MERN stack workflow** including authentication, state management, and REST API integration.

## Tech Stack

**Frontend**
- React
- React Router
- Context API
- Tailwind CSS

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose

**Other Tools**
- Git
- GitHub
- REST APIs

---

## Features

- User authentication (Register / Login)
- Restaurant listing and menu browsing
- Cart management
- Order placement
- Order history
